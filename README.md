# 📊 Auditoría de Integridad y Performance - E-commerce Olist (Brasil)

## 🎯 Objetivo del Proyecto
Analizar la consistencia entre las transacciones de venta y los registros de pagos para detectar fugas de capital e ineficiencias logísticas. 

## 🧠 Enfoque Contable
Como estudiante de **Contador Público (UNC)**, apliqué criterios de auditoría financiera para:
* **Validación de Integridad:** Crucé de bases de datos para identificar órdenes entregadas sin registro de cobro.
* **Análisis de Performance:** Cálculo de *Lead Time* y detección de cuellos de botella en la entrega.
* **Análisis de Pareto:** Identificación del 20% de categorías que generan el 80% del ingreso.

## 🛠️ Stack Técnico
* **Python** (Pandas para limpieza y unión de datasets).
* **Matplotlib/Seaborn** (Visualización de tendencias).
* **Jupyter Notebooks** para el análisis exploratorio.

## 📈 Hallazgos Clave
* Se detectó un **4% de órdenes con demoras críticas** (>30 días), impactando en la retención de clientes.
* Identificación de inconsistencias en registros de pagos en un X% de las transacciones (fugas potenciales).
